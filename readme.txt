login.py 是一个python脚本，主要用来对一个登陆网站用户名密码的爆破。
初衷：
很多页面登陆，有弱密码，这个脚本也就是来检测弱密码的。所以，用户名以及密码都不会太多，这样就只是检测了一下
弱密码，当然，如果用户需要，也可以指定一个大的密码库来爆破密码。

目前第一版，只是对没有验证码的进行了爆破。也就是，登陆界面就是用户名密码两个input。并且服务端不会对该页面
登陆次数进行限制。

2015-1-29:版本进行了一次简单的跟新，用户可以自己指定提交的url以及用户名密码别称。这为了解决一下可能分析不出来的
一些情况。比如html5做的一些登陆界面。这就需要用户有较强的能力去分析。
改动在login.py里面，改动不大。

流程：
第一，得到登陆url，例如http://www.test.com/login.html。解析这个页面，得到form表单的提交的地址，提交的方式
提交的参数名。
第二，根据域名，按照规则生成用户名以及密码，同时一些常用的用户密码（弱密码）也添加进去。
第三，还可以自己指定用户密码文件。
第四，用户可以指定错误码，比如登陆不成功的关键字。这样登陆成功与否会根据用户错误关键字判断，否则，程序将会自动
根据返回的内容的长度进行判断。
第五，如果登陆之后是跳转，程序也做了判断，跳转之后再进一步进行爆破。
第六，建立一个线程池，可以自己指定线程池的大小。每一个线程进行用户密码爆破。
第七，得到结果

特点：
1、根据url生成用户米密码。
2、自己可以添加用户密码的文档
3、可自己配置线程池
4、用户自己配置错误关键字
